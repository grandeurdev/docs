# Graph

The Graph widget displays quantitative attributes from a data source as a graphical representation. It allows end users to visualize and observe possible patterns and trends from raw data.

| Parameter     | Description|
| :------------ | ---------: |
| Title        |    Name the widget by choosing a title of your choice.  | 
| Variable     |    Choose the variable to read from.    |
