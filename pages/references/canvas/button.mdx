# Button

Make buttons and control other services with a single tap. The button widget allows to send commands and view binary statuses of the device, such as power ON/OFF.

| Parameter     | Description|
| :------------ | ---------: |
| Title        |    Name the widget by choosing a title of your choice.    | 
| Variable     |    Choose the variable to control/ read from.    |   
| ON/OFF state values |    Set the required values in the corresponding fields. |   
| Push Mode |  Sends on/off state value. |
