# Slider

Slider allows to view and send values in the predefined range. Usage examples: resistance, brightness etc. Set the slider handle to choose the required value.
When the state of the slider changes over a specified range, the widget calls the onChanged callback and updates its status accordingly. 

| Parameter     | Description|
| :------------ | ---------: |
| Title        |    Name the widget by choosing a title of your choice.    | 
| Variable     |    Choose the variable to read from.    |   
| Min Value |    Set min value of certain parameters as per custom needs |   
| Max Value |    Set max value of certain parameters as per custom needs |
