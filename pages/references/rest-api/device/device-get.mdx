# /device/get

### Body:

`{ deviceID: string , path: string  }`

### Success response:

`{code: "DEVICE-DATA-FETCHED", message: "Data for the device is successfully fetched: ", data: "DRONE-001" }`

### Error response:

`{ code: "DATA-INVALID", message: "Both Device and User IDs are required." }`

`{ code: "DEVICE-ID-INVALID", message: "Device is not registered with the project."}`

`{code: "DEVICE-NOT-PAIRED", message: "Device is not paired with the user."}`

`{ code: "DEVICE-UNAUTHORIZED", message: "Device is not authorized to update data of other devices." }`

`{ code: "DATA-INVALID", message: "Device ID is required for fetching its data."}`

`{ code: "PATH-INVALID", message: "Data is undefined at the specified path."}`


### Usage:

```jsx
fetch("https://api.grandeur.dev/projects/meta?apiKey=<APIKEY>",
	{
		headers: {
			"Content-Type": "application/json"
			"gt-otp": SECRET
			Authorization: TOKEN
		}
		body: {
            "deviceID": "{deviceID}",
            "path": ""
        }
	}
);
```
