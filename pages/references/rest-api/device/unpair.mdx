# /device/unpair

### Body:

`{ deviceID:string }`

### Success response:

`{ code: "DEVICE-UNPAIRED", message: "Device is successfully un-paired from the user account." }`

### Error response:

`{ code: "DEVICE-UNPAIRING-FAILED", message: "Device could not be un-paired from the user account." }`

### Usage:

```jsx
fetch("https://api.grandeur.tech/projects/meta?apiKey=<APIKEY>",
	{
		headers: {
			"Content-Type": "application/json"
			"gt-otp": SECRET
			Authorization: TOKEN
		}
		body: {
            "deviceID": "{deviceID}"
        }
	}
);
```
