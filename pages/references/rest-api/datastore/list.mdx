# /datastore/list

### Body:

#### `nPage`

- **Type:** Integer
- **Examples:** `1{:js}`, `10{:js}`

___

### Success response:

#### code

`"DATASTORE-COLLECTIONS-FETCHED" {:js}`

#### message

`"List of collections is successfully fetched from the datastore." {:js}`

#### nCollections

- **Type:** Integer
- **Examples:** `5 {:js}`

#### collections

- **Type:** List
- **Examples:** `[{ collectionID: "ck61726912hdkjhd802809109j", nDocuments: 10 }] {:js}`

___

### Error response:

#### `DATASTORE-COLLECTIONS-FETCH-FAILED`

`{ code: "DATASTORE-COLLECTIONS-FETCH-FAILED", message: " List of collections could not be fetched from the datastore." } {:js}`

___

### Usage:

<Tabs items={['cURL', 'Javascript', 'Python']}>
<Tab>

```bash
curl "https://api.grandeur.tech/datastore/list?apiKey=${API_KEY}" -H "Authorization: ${AUTH_TOKEN}" --json '{"nPage": "1"}'
```

</Tab>
<Tab>
```jsx
fetch(`https://api.grandeur.tech/datastore/list?apiKey=${API_KEY}`,
    {
        method: "POST",
        headers: {
            "Content-Type": "application/json"
            Authorization: AUTH_TOKEN
        }
        body: {
            nPage: 1                    // First 20 collections
        }
    }
)
.then(console.log);
```
</Tab>
<Tab>

```py
import requests

response = requests.post(
    f'https://api.grandeur.tech/datastore/list?apiKey={API_KEY}',
    headers={'Authorization': AUTH_TOKEN},
    json={"nPage": 1}
)

print(response.text)
```
