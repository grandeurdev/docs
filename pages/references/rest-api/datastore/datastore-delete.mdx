# /datastore/delete'

### Body:

`{ collection:string , filter:object }`

### Success response:

`{ code: "DATASTORE-DOCUMENTS-DELETED", message: "Documents are successfully deleted from the datastore."}`

### Error response:

`{code: "DATA-INVALID", message: "Collection name is required to be specified."}`

`{code: "DATASTORE-DOCUMENTS-NOT-FOUND", message: "No document in the datastore satisfies the query."}`

`{code: "DATASTORE-DOCUMENTS-DELETION-FAILED", message: "An unknown error occurred while deleting from the datastore."}`

### Usage:

```jsx
fetch("https://api.grandeur.dev/projects/meta?apiKey=<APIKEY>",
	{
		headers: {
			"Content-Type": "application/json"
			"gt-otp": SECRET
			Authorization: TOKEN
		}
		body: {
            "collection": "name",
            "filter": {}
        }
	}
);
```
