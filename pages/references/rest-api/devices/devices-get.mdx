# /devices/get

### Body:

`{ type:string , filter:object , productID: string }`

### Success response:

`{ code: "DEVICES-LIST-FETCHED", message: "List of requested devices is successfully fetched", nDevices: 1, 
devices: [ { deviceID: "device162612983dhqdh982iasu092", productID: "DRONE-1", status: false, admin: "tonystark@grandeur.dev", data: { state: 0, current: 10 } } ]}`

### Usage:

```jsx
fetch("https://api.grandeur.dev/projects/meta?apiKey=<APIKEY>",
	{
		headers: {
			"Content-Type": "application/json"
			"gt-otp": SECRET
			Authorization: TOKEN
		}
		body: {
            "type": "registered/paired/unpaired",
		    "filter": "online/offline",
            "productID": ""
        }
	}
);
```
