# Overview

This section guides you on how to set up the hardware board you are working with, including any circuitry or core and library installations that are required to program your board.

The list is non-exhaustive and more and more boards are supported on weekly basis. If you can't find your board, [let us know](mailto:hi@grandeur.dev).

2. [ESP32](setting-up-the-board/setting-up-esp32)
