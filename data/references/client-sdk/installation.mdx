# Installation


<Tabs group="tab-group-test">

  <Item value="js" label="JS">

Simply drop the link of JavaScript SDK in a script tag inside your web app using our CDN.


  </Item>

  <Item value="react" label="React">

   Simply download the package from npm and use it in your react project. It is as shown below
  </Item>

</Tabs>


<Tabs group="tab-group-test">

  <Item value="js" label="JS">

 ```js
    <!-- Drop the Link of CDN in your Web App -->
    <script src="https://unpkg.com/grandeur-js"></script>
```


  </Item>

  <Item value="react" label="React">

    ```js
    npm install grandeur-react--save
    ```


</Item>
