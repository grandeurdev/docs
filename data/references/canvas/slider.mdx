#Slider


Slider allows to view and send values in the predefined range. Usage examples: volume, brightness etc. Set the slider handle to choose the required value.
When the state of the slider changes over a specified range, the widget calls the onChanged callback and updates its status accordingly. 

-Title: Name the widget by choosing a title of your choice.

-Variable: Choose the variable to control/read from.

-Min/Max Values: Set min and max values of certain parameters as per custom needs
