# Progress

Displays the current progress over a specified range. The progress is displayed as a %age.

| Parameter     | Description|
| :------------ | ---------: |
| Title        |    Name the widget by choosing a title of your choice.    | 
| Variable     |    Choose the variable to read from.    |   
| Min/Max Values |    Set min and max values of certain parameters as per custom needs |
